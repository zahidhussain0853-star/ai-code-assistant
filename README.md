# 🧠 AI Code Assistant

A web-based AI coding assistant built with Flask and OpenRouter API.

## 🚀 Features
- Generate code
- Explain code
- Fix bugs
- Interactive chat UI

## 🛠 Tech Stack
- Python (Flask)
- HTML/CSS
- OpenRouter API

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
